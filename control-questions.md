# Ответы на контрольные вопросы
## 1. Какая текущая версия MacOS?
MacOS 26 Tahoe
## 2. На основе какой UNIX-подобной системы построена операционная система macOS?
Операционная система MacOS построена на основе Darwin
## 3. К какому типу относится ядро macOS? В чем отличия между XNU и Mach?
Ядро macOS (XNU) относится к гибридному типу (hybrid kernel). XNU - полное ядро операционной системы, а Mach - его компонент
## 4. Какой тип архитектуры у процессоров линейки M1 и выше, производимых Apple?
Процессоры линейки Apple M1, M2, M3 и выше имеют архитектуру ARM
## 5. Что такое виртуализация? Приведите примеры гипервизоров.
Виртуализация — это технология, которая позволяет создавать виртуальные (программные) версии компьютерных ресурсов: операционных систем, серверов, хранилищ данных, сетей и т.д.
Примеры гипервизоров:
1. VMWare ESXi
2. Microsoft Hyper-V
3. Oracle VirtualBox
